# MasculinePath — Landing Page

Static landing page. Pure HTML, CSS, and vanilla JS — no build step.

## Files
- `index.html` — page markup
- `style.css` — design system & responsive styles
- `script.js` — sticky nav, mobile menu, scroll reveals, FAQ accordion

## Deploy
Push to GitHub and serve via GitHub Pages, Netlify, Vercel, or any static host. No dependencies.

## Customize
- Replace `$X/month` with your price.
- Replace `#` in CTA links with real URLs (membership signup, booking page).
- Update Instagram / X links in the footer.
